# 内容结构规划 - 简洁电商购物网站

## 1. 材料清单

**设计依据：**
- `docs/jd_analysis.md`（京东电商分析文档）- 功能模块、用户流程、交互模式洞察
- 风格指南：E-commerce Optimized（电商转化优化型）

**设计目标：**
- 移除所有广告、弹窗、浮层等干扰元素
- 聚焦核心购物功能：搜索、浏览、比较、购买、管理
- 采用纯白背景 + 大图展示 + 高对比CTA的转化优化设计

## 2. 网站结构

**类型：** MPA（多页面应用）

**理由：** 
- 7大独立页面类型，各自承担不同购物任务
- 功能完整（商品分类、搜索、购物车、用户中心、订单管理等）
- 内容多样性高（商品列表、详情、用户数据、订单历史）
- 符合电商标准架构（SEO友好，分享友好）

## 3. 页面/功能模块划分

### 页面 1: 首页（Home） `/`

**目的：** 快速发现商品 + 开始购物旅程

**内容映射：**

| 区块 | 组件模式 | 内容来源 | 提取内容 | 视觉素材 |
|------|----------|----------|----------|----------|
| 导航栏 | E-commerce Header | 标准导航 | Logo、搜索框、购物车、用户入口 | Logo图标 |
| Hero区 | Hero Banner | 动态推荐 | 主推商品/活动（单个，无轮播） | 高质量产品图 |
| 分类导航 | Category Grid | 标准分类 | 8-12个主分类（图标+文字） | 分类图标（SVG） |
| 热门商品 | Product Grid | 推荐算法 | 8-16个热销商品卡片 | 商品图片（1:1） |
| 新品推荐 | Product Grid | 商品数据库 | 8-16个新品卡片 | 商品图片（1:1） |
| 底部 | Footer | 标准信息 | 帮助、政策、联系方式 | - |

**关键特性：**
- Hero区仅展示1个主推，无自动轮播（京东分析洞察：轮播降低注意力）
- 商品网格3-4列（桌面），信息密度高
- 搜索框占据导航栏40-50%宽度（突出检索入口）

### 页面 3: 商品详情页（Product Detail） `/product/:id`

**目的：** 深度展示商品信息 + 决策支持 + 完成加购

**内容映射：**

| 区块 | 组件模式 | 内容来源 | 提取内容 | 视觉素材 |
|------|----------|----------|----------|----------|
| 导航栏 | Sticky Header | 标准 | 固定顶部 | - |
| 面包屑 | Breadcrumb | 分类路径 | 返回上级分类 | - |
| 图片画廊 | Product Gallery | 商品图片库 | 4-6张高清图（主图+细节+使用场景） | 商品多角度图片 |
| 购买盒子 | Sticky Buy Box | 商品数据 | 标题、评分、价格、库存、规格、加购按钮 | - |
| 商品详情 | Tabbed Content | 商品描述 | 详细描述、参数表、尺码指南 | 细节图片 |
| 评价系统 | Review Section | 用户评价 | 评分、评价列表、筛选（好评/差评） | 用户上传图片 |

**关键特性：**

- 价格显眼：当前价（红色、大号、粗体） + 原价（删除线）
- 库存提示（"仅剩5件"）营造紧迫感（克制使用，非弹窗）
- 评价维度化（质量、价格、物流）+ "有用"投票

## 4. 内容分析

**信息密度：** 中高密度
- 商品列表页：密集展示（3-4列网格），高效浏览
- 商品详情页：信息完整但模块化（图片、价格、评价清晰分组）
- 结算流程：表单精简，必填项最少

**内容平衡：**
- 图片：60%（商品图为核心）
- 数据：30%（价格、库存、评分、订单信息）
- 文本：10%（描述、评价、政策）
- 内容类型：视觉驱动 + 数据支撑型

**功能完整性：**
- ✅ 商品发现（首页、分类、搜索）
- ✅ 商品决策（详情页、评价、规格对比）
- ✅ 购买流程（加购、购物车、结算、支付）
- ✅ 订单管理（订单列表、详情、物流、售后）
- ✅ 用户功能（账户、收藏、优惠券、地址）
- ❌ 移除内容：广告、弹窗、浮层、自动轮播、强制引导

**设计哲学：**
- 京东洞察应用：去噪增效、筛选优先级、评价维度化、结算精简
- 转化优化：大图展示、价格显性、库存提示、高对比CTA
- 信任建立：评价系统、物流透明、售后明确、安全标识

---

## 5. 关键设计决策

### 5.1 去除的干扰元素（基于京东分析）

❌ **移除项：**
- 首页轮播广告（注意力分散）
- 弹窗/浮层（中断任务）
- 活动H5强曝光（界面噪声）
- 侧边栏运营位（占据空间）
- 自动播放视频（性能影响）
- 过多榜单穿插（从众压力）

### 5.2 保留的必要元素（转化支撑）

✅ **保留项：**
- 价格信息（透明、显眼）
- 库存提示（"仅剩X件"，克制使用）
- 评分/评价（社会证明）
- 优惠券提示（价值感知）
- 物流承诺（"包邮"、"次日达"）
- 信任标识（"7天退换"、"正品保证"）

### 5.3 交互优化（效率提升）

✅ **优化策略：**
- 搜索联想（关键词→分类→店铺优先级）
- 快速加购（悬停按钮）
- 筛选记忆（用户偏好保存）
- 一键结算（表单自动填充）
- 物流透明（实时更新）
